# webcodec-showcase
## What
* It a low-level `web API` for handling media
* It gives developers control to encode and decode audio and video directly in the browser, enabling advanced features like real-time editing, transcoding, conferencing, and streaming without complex external libraries, by providing access to built-in hardware/software codecs for tasks like
* It allows fine-grained control over media frames, integrates with WebRTC and WebGPU, and offloads heavy processing to Web Workers for better performance.

## When
* WebCodecs support rolled out gradually, with major browsers adding full support around 2021-2022, while Safari followed with VideoDecoder and is adding AudioDecoder in its newer versions (in late 2025), making it widely available across major platforms now for low-level media processing.
